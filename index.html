About

Ever since I saw what computers and software could achieve as a young kid, I've been interested in technology. My interest in software development was later fueled by the discovery of the popular Lego Mindstorms robotics kits. This learning tool gave me the motivation to study for a bachelor's in computer science.

In the time since college, I have gained experince as a webmaster and a robotics sales assistant. Recently, I took part in the Cook Systems Fasttrack Program, where I learned how to use Java with Spring Boot to develop web applications. I have a few hobbies, which include: writing software, 3d printing, and spending time with friends or family.




Work

Quiz API

Making use of a basic code skeleton, I implemented an application that enabled users to manipulate quizzes containing a series of questions and answers. Prime examples of the functions I worked on would be creating quizzes, retrieving quizzes, getting random questions, and deleting questions. It served as a valuable first step into designing web applications.

Environment: Java, Spring Boot, Spring JPA

Social Media API

In a team environment, I helped develop the core functions of a social media platform from scratch. Some notable features include creating tweets, adding hashtags, following other users, and viewing a personalized feed. Thanks to a comprehensive test suite, it has been refined to the point that it could be used as a basis for a full-fledged social media platform.

Environment: Java, Spring Boot, Spring JPA, Maven
